## 二、申请步骤（个人账号）

### 第1步：注册阿里云账号（5分钟）

1. 访问：https://free.aliyun.com/
2. 点击右上角"免费注册"
3. 选择"个人用户"
4. 填写：
   - 手机号
   - 验证码
   - 设置密码
5. 点击"同意协议并注册"

### 第2步：实名认证（5分钟）

注册后会自动跳转到实名认证页面：

1. 选择"个人实名认证"
2. 填写身份信息：
   - 真实姓名
   - 身份证号
   - 上传身份证照片（正反面）
3. 人脸识别验证（支付宝扫码）
4. 等待审核（通常5分钟内通过）

**注意：必须实名认证才能申请免费试用！**

### 第3步：领取免费试用资源（3分钟）

1. 回到 https://free.aliyun.com/
2. 找到"云服务器ECS"产品卡片
3. 点击"立即试用"
4. 选择配置：
   ```
   地域：华东1（杭州）或华北2（北京）- 就近选择
   操作系统：Ubuntu 22.04 64位
   实例规格：ecs.t5-lc1m2.small（2核2G）
   试用时长：3个月
   ```
5. 勾选"我已阅读并同意服务协议"
6. 点击"立即试用"

### 第4步：创建服务器实例（5分钟）

1. 试用成功后，进入"云服务器ECS控制台"
2. 点击"创建实例"
3. 选择配置：
   ```
   计费模式：按量付费（试用期免费）
   地域：华东1（杭州）
   实例规格：ecs.t5-lc1m2.small
   镜像：Ubuntu 22.04 64位
   网络：默认专有网络VPC
   公网IP：分配（免费5Mbps带宽）
   安全组：默认安全组
   ```
4. 设置登录密码：
   ```
   用户名：root（系统默认）
   密码：自己设置一个复杂密码（大小写+数字+符号，记住它！）
   ```
5. 点击"确认创建"

### 第5步：配置安全组（重要！）

必须开放端口，否则无法访问：

1. 进入ECS控制台
2. 点击实例右侧"更多" → "网络和安全组" → "安全组配置"
3. 点击"配置规则" → "入方向" → "手动添加"
4. 添加以下规则：

| 端口范围 | 授权对象 | 说明 |
|---------|---------|------|
| 5000/5000 | 0.0.0.0/0 | Flask主界面 |
| 8501/8501 | 0.0.0.0/0 | Streamlit Agent1 |
| 8502/8502 | 0.0.0.0/0 | Streamlit Agent2 |
| 8503/8503 | 0.0.0.0/0 | Streamlit Agent3 |
| 22/22 | 0.0.0.0/0 | SSH远程登录 |

5. 点击"保存"

## 四、测试连接（可选）

如果想自己先测试一下服务器能否连接：

**Windows用户：**
1. 下载 XShell 或 PuTTY
2. 输入服务器IP和密码
3. 能登录就说明成功

**Mac用户：**
1. 打开终端
2. 输入：`ssh root@你的服务器IP`
3. 输入密码
4. 能登录就说明成功

## 五、常见问题

### Q1：免费试用到期后怎么办？
**A：**
- 试用3个月后，可以转为按量付费（大约100-200元/月）
- 或者用其他人的阿里云账号重新申请试用
- 或者迁移到本地电脑运行

### Q2：个人账号和企业账号能同时试用吗？
**A：**
- 不能，一个身份证只能试用一次
- 但可以用公司营业执照再申请一个企业账号试用

### Q3：试用期间会扣费吗？
**A：**
- 试用期内服务器免费
- 但如果超出流量限制（5Mbps带宽），会产生少量流量费
- 我们这个系统流量很小，基本不会超

### Q4：需要备案吗？
**A：**
- 不需要！我们只是内部使用，不对外公开网站
- 只有对外提供网站服务才需要备案

### Q5：服务器在哪个地区比较好？
**A：**
- 华东1（杭州）或华北2（北京）都可以
- 选离你公司近的，访问速度更快
